一个使用 C++ 和 Python 混合开发的控制器项目。

## 项目概述

本项目是一个功能完整的控制器系统，采用以下技术栈：

- **C++** (51.9%) - 核心系统逻辑和高性能计算
- **Python** (46.8%) - 业务逻辑和脚本控制
- **PowerShell** (1.3%) - 构建和部署脚本

## 主要特性

- ✨ 高性能的 C++ 核心实现
- 🐍 灵活的 Python 接口
- 🔧 自动化的构建和部署流程
- 📦 模块化设计

## 快速开始

### 前置要求

- C++17 或更高版本
- Python 3.7+
- CMake（推荐）

### 安装

```bash
git clone https://github.com/kevin07429/Controller.git
cd Controller
```

### 编译

```bash
# 使用 CMake
mkdir build
cd build
cmake ..
make
```

### 使用方法

```bash
# 运行您的应用
./Controller
```
